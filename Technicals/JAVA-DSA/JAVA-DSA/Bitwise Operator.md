## **Left Shift `<<`**
- (10) of base 10 = (1010) of base 2 
- if, 10 << 1 => (10100) of base 2 => (20) of base 10.
- `a << 1 = 2a`
- `a << n = a * 2^b`
## **Right Shift** `>>`
- 00011001 >> 1 = 001100 => remove the last digit.
- leading zeros are ignored for all number system(decimal, binary, octal, hexadecimal).
- `a >> n = a/2^b`
